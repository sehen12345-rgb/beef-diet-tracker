# 소고기 식단 트래커 — 인스타 + ManyChat 마케팅 전략

---

## 1. 전체 퍼널 구조

```
인스타 콘텐츠 노출
    → 댓글/DM 키워드 트리거 (ManyChat)
        → 자동 DM: 무료 다운로드 링크
            → 앱 설치 (14일 체험)
                → D+12 자동 DM: 유료 전환 유도
                    → 결제 (litt.ly / 토스페이먼츠)
                        → 라이선스 키 발급 → 앱 활성화
```

---

## 2. 인스타그램 콘텐츠 전략

### 피드 (1:1, 1080×1080)
- **주 2회** 정기 포스팅
- 내용: 앱 스크린샷 + 실제 식단 인증 카드 (앱에서 바로 생성)
- 캡션 예시:
  ```
  오늘 단백질 달성률 94% 💪
  
  소고기 홍두깨살 200g으로
  하루 단백질 목표를 거의 다 채웠어요 🥩
  
  👇 무료 트래커 받는 법
  댓글에 "무료" 남겨주세요!
  DM으로 다운로드 링크 보내드려요
  
  #소고기식단 #비프다이어트 #단백질챌린지
  #다이어트식단 #클린이팅 #근성장 #헬스낵
  ```

### 스토리 (9:16, 1080×1920)
- **매일** 오늘의 식단 인증 (앱 공유 기능으로 자동 생성)
- 스티커: "DM으로 무료 받기 👆"
- 하이라이트: "식단 트래커" 로 묶기

### 릴스 (9:16)
- 월 2회 제작
- 내용: "하루 소고기 식단 브이로그" / "50일 챌린지 중간 결과"
- 첫 3초: 충격적인 비포/애프터 (체중 변화 그래프)
- 마지막: "댓글에 '무료' → 앱 다운로드 링크 DM"

## 3. ManyChat 자동화 설정

### 설치
1. manychat.com → Instagram 연결
2. 무료 플랜으로 시작 (월 1,000 연락처까지 무료)

### 플로우 1: 댓글 → DM 자동 응답

**트리거**: 특정 포스트 댓글에 "무료" 포함  
**액션**:
```
DM 자동 발송:
"안녕하세요! 🥩 소고기 식단 트래커 무료 버전 링크예요.

👇 다운로드
[litt.ly 링크]

14일 무료 체험 후 프리미엄으로 업그레이드하면
워터마크 없는 인스타 공유 카드를 만들 수 있어요!

사용 중 궁금한 점은 DM 주세요 😊"
```

### 플로우 2: D+12 전환 유도 시퀀스

**트리거**: 무료 링크 클릭 후 12일 뒤 (ManyChat 시퀀스)  
**액션**:
```
Day 12 DM:
"🥩 소고기 식단 트래커 사용 중이신가요?

체험 기간이 2일 남았어요!

지금 업그레이드하면:
✅ 인스타 공유 카드 워터마크 없음
✅ 모든 기능 무제한
✅ 향후 업데이트 무료

👉 지금 구매하기: [litt.ly/beeftracker]

오늘만 할인 코드 BEEF10 사용하면 10% 할인!"
```

**Day 14 DM** (체험 만료일):
```
"⏰ 오늘로 무료 체험이 종료됩니다.

50일 챌린지 기록이 사라지기 전에
업그레이드하세요!

→ [litt.ly/beeftracker]"
```

### 플로우 3: 구매 완료 → 키 자동 발급

**트리거**: litt.ly 구매 완료 웹훅  
**액션**:
1. 구매자 이메일로 `python generate_key.py 이메일` 실행
2. 라이선스 키를 ManyChat으로 DM 발송
```
"🎉 프리미엄 구매 감사합니다!

라이선스 키: BEEF-XXXX-XXXX-XXXX

앱 실행 → 우측 상단 🔑 아이콘 → 키 입력
모든 기능이 활성화됩니다!

문의: DM 또는 @beeftracker_kr"
```
